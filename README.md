# Similarity Score Test

이 프로젝트는 그림 유사도 측정 및 Google QuickDraw 데이터셋을 활용한 스트로크 데이터 처리/시각화를 위한 프로젝트입니다.

Gemini와 Claude Code를 활용하여 제작되었습니다.

## 설치 및 실행 방법 (Installation & Usage)

### 1. 설치 (Installation)
루트 경로와 클라이언트 경로 모두에서 패키지 설치가 필요합니다.

**루트 폴더 (데이터 처리 스크립트용)**
```bash
npm install
```

**클라이언트 폴더 (웹 앱용)**
```bash
cd client
npm install
```

### 2. 실행 (Run)
클라이언트 애플리케이션을 실행하여 웹 브라우저에서 확인합니다.
```bash
cd client
npm run dev
```

## 주요 기능 설명

### 1. 그림 그리기 및 데이터 확인
- **게임 탭**: 캔버스에 직접 마우스나 터치로 그림을 그릴 수 있습니다.
- **데이터 확인**: 하단의 `디버깅 정보 (스트로크 데이터)`를 클릭하면 현재 캔버스에 그려진 그림의 스트로크 데이터(좌표, 색상 정보)를 실시간으로 확인할 수 있습니다.

### 2. 유사도 점수 테스트
- **실시간 점수 산출**: 게임 탭에서 왼쪽의 '제시 그림'을 따라 오른쪽 '내 그림' 캔버스에 그리면, 즉시 유사도 점수가 계산되어 화면 중앙에 표시됩니다.
- **스트로크 데이터 로드 테스트**: 
    1. 미리 준비된 JSON 형식의 스트로크 데이터를 `스트로크 데이터 로드` 텍스트 영역에 입력합니다.
    2. `스트로크 데이터 로드` 버튼을 클릭하면 해당 데이터가 캔버스에 렌더링됩니다.
    3. 동시에 해당 그림에 대한 유사도 점수가 산출됩니다.

### 3. QuickDraw 데이터 랜덤 다운로드
Google QuickDraw 데이터셋에서 실제 사람이 그린 그림 데이터를 가져와 프로젝트 데이터로 활용할 수 있습니다.

**사용 방법**
루트 디렉토리에서 다음 명령어를 실행합니다.
```bash
node enrich_data.js
```

**동작 원리**
1. Google QuickDraw의 전체 카테고리 중 랜덤으로 몇 가지를 선정합니다.
2. 선정된 카테고리에서 `recognized: true`인(정답으로 인식된) 그림 100개를 랜덤 추출합니다.
3. 원본 데이터에는 색상 정보가 없으므로, 지정된 팔레트 내에서 랜덤한 색상을 각 스트로크에 부여하여 변환합니다.
4. 변환된 데이터를 `drawing.json` 파일에 추가(append)합니다.

**데이터 확인**
- 웹 앱 상단의 **데이터 미리보기** 탭으로 이동하면, 추가된 데이터를 포함한 모든 그림 데이터를 그리드 형태로 미리볼 수 있습니다.
